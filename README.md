# SMK-Coding-Project_2
Tugas dari SMK Coding Tahap 2 bertemakan monitoring COVID-19 berdasarkan data nasional, provinsi, dan dunia menggunakan : 
* ViewPager
* Fragment
* Public REST Api
* SharedPreferences
